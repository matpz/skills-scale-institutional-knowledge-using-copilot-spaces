# OctoAcme Project Management Documentation

Welcome to the OctoAcme project management process hub. This folder contains standardized documentation for how we run projects, manage risks, and deliver value.

## Overview of OctoAcme Project Management

OctoAcme follows a structured, lifecycle-based approach to project management that emphasizes customer value, iterative delivery, and clear ownership. The process is organized around five key phases: **Initiation**, **Planning**, **Execution**, **Release**, and **Close & Retrospective**. 

Our methodology prioritizes validating business needs upfront, breaking work into shippable increments, embedding quality throughout the delivery cycle rather than at the end, and institutionalizing continuous improvement through retrospectives. Communication is structured through daily standups, weekly delivery syncs, and monthly stakeholder updates, with clear escalation paths for blocked issues. Success is measured through defined metrics captured in Project One-pagers, and every team member—from developers to project managers—understands their role in delivering customer value.

## Core Principles

- **Customer-first**: Prioritize customer value and usability
- **Iterative delivery**: Deliver small, testable increments
- **Clear ownership**: Each project has a named Project Manager and Product Lead
- **Data-informed**: Measure impact and iterate based on evidence
- **Psychological safety**: Encourage feedback and learning

## Key Artifacts

- Project Charter / One-pager
- Backlog and Release Plan
- Risk Register
- Sprint/Iteration Plans
- Release Notes
- Retrospective notes and action items

## Communication Cadence

- **Daily standups** — 15 minutes, focus on progress, blockers, dependencies
- **Weekly PM sync** — Alignment between Project Manager and Product Lead
- **Twice-weekly delivery standups** — Team-level progress and coordination
- **Monthly stakeholder updates** — High-level status for sponsors and stakeholders
- **Ad-hoc escalations** — Following the escalation ladder: team → PM → Product Lead → Sponsor
